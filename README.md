## 🚀 Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/contaboo.git
cd contaboo
```

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to see the application.
